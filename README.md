# MartixOS

**Version:** 0.1 (Beta)  
**License:** GPLv3  
**Author:** Martin Yanchev  
**Purpose:** Minimal and flexible Linux distribution, designed for gaming and development, with a focus on Wayland and modern tools.

---

## Project Goal

MartixOS is created with the goal to be:

- Lightweight and minimal, free from unnecessary software bloat  
- Fully Wayland-based for modern display management  
- Ready for gaming and development out of the box  
- Open-source under GPLv3, ensuring freedom and transparency  

---
